# Project Setup

```bash
# Clone this repository
$ git clone https://github.com/laboratorio-de-praticas/cms-fe.git

# Go into the repository
$ cd cms-fe
```

# Install and compile

```bash
# Install dependencies
$ npm install or npm i

# Run the app
$ npm run dev
```
